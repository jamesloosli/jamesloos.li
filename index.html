Automation

Salt

Saltstack is my current preferred framework for instance lifecycle, configuration, and orchestration. I've thrown together a small 'quickstart' demo for using saltstack in what I feel is an extensible way.

Puppet

While I'm not a current Puppet user, I did use it exclusively for 3 years. Puppet has a much better sdlc than SaltStack does when it comes to modularity and usability. For larger environments, Puppet would still be my first choice.

Clouds

Amazon Web Services

I'm an AWS fan, and I can navigate the sea of available options pretty well. My preference is to build infrastructure with as thin a requirement on specific services as possible, but I am familiar with things like CodeDeploy, CodePipeline, Elasticache, RDS etc.

Digital Ocean

For small projects that don't need the comprehensive featureset AWS offers, Digital Ocean is hard to beat. I've run infrastructure for multiple companies in DO, and found them great to work with.
